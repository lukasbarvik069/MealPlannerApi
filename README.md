# Meal Planner API (Ve vývoji)

Backendové rozhraní pro aplikaci na plánování stravování. Projekt je momentálně ve fázi vývoje API rozhraní.

## Aktuální stav architektury
- **Models:** Databázové entity reprezentující strukturu dat v EF Core.
- **DTOs (Data Transfer Objects):** Samostatné objekty pro přenos dat mezi klientem a serverem, zajišťující bezpečnost a optimalizaci.
- **API Controllers:** RESTful endpointy pro manipulaci s daty (Recepty, Jídelníčky).

## Technologie
- **.NET 8** (C#)
- **ASP.NET Core Web API**
- **Entity Framework Core**
